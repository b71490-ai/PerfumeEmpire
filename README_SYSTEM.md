## 🎯 الأقسام الرئيسية

### 1️⃣ **واجهة العميل**

#### الصفحات المتاحة

- 🏠 **الصفحة الرئيسية:** `/`
  - عرض تقديمي جذاب
  - العروض الخاصة
  - الفئات المتاحة

- 🛍️ **المتجر:** `/shop`
  - عرض جميع المنتجات
  - البحث والتصفية
  - إضافة للسلة والمفضلة

- 📦 **تفاصيل المنتج:** `/product/[id]`
  - معلومات تفصيلية
  - إضافة للسلة
  - إضافة للمفضلة

- 🛒 **السلة:** `/cart`
  - عرض المنتجات المضافة
  - تعديل الكميات
  - حساب الإجمالي

- ❤️ **المفضلة:** `/wishlist`
  - قائمة المنتجات المفضلة
  - نقل للسلة

- 📞 **اتصل بنا:** `/contact`
  - نموذج تواصل
  - معلومات الاتصال
  - الأسئلة الشائعة

#### المميزات

- ✅ تصفح المنتجات بدون قيود
- ✅ إضافة للسلة والمفضلة
- ✅ حساب الأسعار مع الخصومات
- ✅ واجهة عربية احترافية
- ❌ **لا يمكن تعديل أو حذف المنتجات**

---

### 2️⃣ **لوحة الإدارة**

#### تسجيل الدخول

- **الرابط:** `/admin/login`
- **كلمة المرور الافتراضية:** `admin123`
- **الحماية:** جميع صفحات الإدارة محمية

#### الصفحات الإدارية

##### 🎛️ **لوحة التحكم الرئيسية** `/admin/dashboard`

- إحصائيات عامة:
  - 📦 عدد المنتجات
  - 🛒 عدد الطلبات
  - 💰 إجمالي المبيعات
- روابط سريعة للإجراءات
- النشاط الأخير

##### 📋 **إدارة المنتجات** `/admin/products`

- عرض جميع المنتجات في جدول
- معلومات كل منتج:
  - الصورة
  - الاسم والفئة
  - السعر والخصم
  - المخزون المتوفر
- إجراءات:
  - ✏️ تعديل المنتج
  - 🗑️ حذف المنتج
  - ➕ إضافة منتج جديد

##### ➕ **إضافة منتج جديد** `/add`

- ⚠️ **محمية - للإدارة فقط**
- نموذج شامل:
  - اسم المنتج
  - العلامة التجارية
  - السعر
  - رابط الصورة
  - نسبة الخصم
  - الفئة (رجالي/نسائي/بخور/تجميل)
- التحقق من البيانات
- رسائل نجاح/فشل

##### ✏️ **تعديل منتج** `/edit/[id]`

- ⚠️ **محمية - للإدارة فقط**
- تحميل بيانات المنتج الحالية
- تعديل أي حقل
- حفظ التغييرات
- إمكانية الحذف

##### 📦 **إدارة الطلبات** `/admin/orders`

- عرض جميع الطلبات
- تفاصيل كل طلب
- حالة الطلب
- متابعة التوصيل

## 🔐 نظام الحماية والصلاحيات

### للعميل

- ✅ التصفح والمشاهدة
- ✅ الإضافة للسلة والمفضلة
- ❌ التعديل أو الحذف

### للإدارة بعد تسجيل الدخول

- ✅ جميع صلاحيات العميل
- ✅ إضافة منتجات جديدة
- ✅ تعديل المنتجات الموجودة
- ✅ حذف المنتجات
- ✅ إدارة الطلبات
- ✅ عرض الإحصائيات

### الحماية المطبقة

1. **AdminContext:** إدارة حالة تسجيل الدخول
2. **حواجز useEffect:** التحقق من الصلاحيات في كل صفحة
3. **إعادة توجيه تلقائية:** توجيه غير المصرح لهم إلى صفحة تسجيل الدخول
4. **عرض شرطي للواجهة:** إخفاء أزرار الإدارة عن العملاء

## 🎨 المميزات التقنية

### واجهة المستخدم (Frontend)

- ⚡ Next.js 13.5.11 (موجّه التطبيق (App Router))
- ⚛️ React 18.2.0
- 🎨 CSS Modules مخصصة
- 🔄 استخدام Context API لإدارة الحالة
- 🔐 نظام حماية متكامل

### الخادم (Backend)

- 🔧 ASP.NET Core 7+
- 🗄️ SQLite Database
- 🌐 CORS مفعّل
- 📡 واجهة برمجية بنمط REST (RESTful API)

### التحليلات والتتبع (GA4 + GTM + Meta Pixel)

- ✅ نظام تتبع موحد عبر ملف `frontend/lib/analytics.js`
- ✅ إرسال متزامن للأحداث إلى:
  - Google Analytics 4 (عبر `gtag`)
  - Google Tag Manager (عبر `dataLayer`)
  - Meta Pixel (عبر `fbq`)

#### إعدادات التتبع من لوحة الإدارة

من صفحة `/admin/settings` في قسم "التحليلات والتتبع":

- `googleAnalyticsId` مثال: `G-XXXXXXX`
- `tagManagerId` مثال: `GTM-XXXXXXX`
- `metaPixelId` مثال: `1234567890`

> عند ترك أي معرّف فارغ، لا يتم تحميل سكربت التتبع المقابل (تحميل شرطي).

#### الأحداث المفعلة حاليًا

- `view_item_list`: عند عرض قائمة المنتجات في المتجر
- `select_item`: عند النقر على منتج من قائمة (المتجر/المفضلة)
- `view_item`: عند فتح صفحة تفاصيل المنتج
- `search`: عند البحث في المتجر
- `add_to_cart`: عند إضافة منتج للسلة
- `remove_from_cart`: عند حذف منتج من السلة
- `view_cart`: عند عرض السلة
- `begin_checkout`: عند بدء إتمام الطلب
- `add_payment_info`: قبل إرسال الطلب
- `purchase`: بعد إنشاء الطلب بنجاح
- `add_to_wishlist`: عند إضافة منتج للمفضلة
- `remove_from_wishlist`: عند إزالة منتج من المفضلة

#### مرجع سريع للبارامترات

- `view_item_list`
  - أساسية: `items[]`, `item_list_name`
  - إضافية: `currency`
  - ملاحظة: يرسل عند عرض قائمة المنتجات في `/shop`

- `select_item`
  - أساسية: `items[]`, `item_list_name`
  - إضافية: `currency`
  - ملاحظة: يرسل عند فتح منتج من قائمة

- `view_item`
  - أساسية: `items[]`, `value`
  - إضافية: `currency`
  - ملاحظة: يرسل عند فتح صفحة المنتج

- `search`
  - أساسية: `search_term`
  - إضافية: `items[]`, `currency`
  - ملاحظة: يرسل عند وجود عبارة بحث فعلية

- `add_to_cart`
  - أساسية: `items[]`, `value`
  - إضافية: `currency`
  - ملاحظة: يرسل عند الإضافة للسلة

- `remove_from_cart`
  - أساسية: `items[]`, `value`
  - إضافية: `currency`
  - ملاحظة: يرسل عند حذف عنصر أو تقليل الكمية إلى الصفر

- `view_cart`
  - أساسية: `items[]`, `value`
  - إضافية: `currency`
  - ملاحظة: يرسل عند عرض السلة

- `begin_checkout`
  - أساسية: `items[]`, `value`
  - إضافية: `currency`
  - ملاحظة: يرسل عند الانتقال من السلة لصفحة الدفع

- `add_payment_info`
  - أساسية: `items[]`, `value`, `payment_type`
  - إضافية: `currency`
  - ملاحظة: يرسل قبل إرسال الطلب

- `purchase`
  - أساسية: `transaction_id`, `items[]`, `value`
  - إضافية: `currency`
  - ملاحظة: يرسل بعد نجاح إنشاء الطلب

- `add_to_wishlist`
  - أساسية: `items[]`, `value`
  - إضافية: `currency`
  - ملاحظة: يرسل عند إضافة منتج للمفضلة

- `remove_from_wishlist`
  - أساسية: `items[]`, `value`
  - إضافية: `currency`
  - ملاحظة: يرسل عند إزالة منتج من المفضلة

> **شكل `items[]` المستخدم:**
> `item_id`, `item_name`, `item_brand`, `item_category`,
> `price`, `quantity`, `discount`.

#### مؤشرات قياس الأداء المقترحة (KPI)

- معدل تحويل قائمة المنتجات → تفاصيل المنتج:
  - `select_item / view_item_list`
- معدل الإضافة للسلة من صفحات المنتج:
  - `add_to_cart / view_item`
- معدل بدء الدفع من السلة:
  - `begin_checkout / view_cart`
- معدل إتمام الشراء:
  - `purchase / begin_checkout`
- معدل نية الشراء عبر المفضلة:
  - `add_to_wishlist / view_item`

#### خريطة الإطلاق حسب الصفحات

- `/shop`
  - `view_item_list`, `search`, `select_item`
  - `add_to_cart`, `add_to_wishlist`, `remove_from_wishlist`
- `/product/[id]`
  - `view_item`, `add_to_cart`, `add_to_wishlist`, `remove_from_wishlist`
- `/wishlist`
  - `select_item`, `add_to_cart`, `remove_from_wishlist`
- `/cart`
  - `view_cart`, `remove_from_cart`, `begin_checkout`
- `/checkout`
  - `add_payment_info`, `purchase`

#### ملاحظات جودة التتبع

- يوجد منع تكرار للأحداث المعرّضة لإعادة الإرسال
  (Deduplication) مثل:
  - `view_item_list`
  - `search`
  - `view_cart`
- تعتمد العملة على إعدادات المتجر (`currencyCode`) وليس قيمًا ثابتة.

#### التحقق السريع بعد التشغيل

1. افتح أدوات المطور في المتصفح وتأكد من وجود `window.dataLayer`.
2. نفّذ إجراءات فعلية (بحث، إضافة للسلة، إتمام طلب).
3. راقب وصول الأحداث في:
   - GA4 DebugView
   - Tag Assistant / GTM Preview
   - Meta Pixel Helper

## 🚨 دليل استجابة الأعطال السريع

### 1) الموقع لا يفتح

- **التحقق السريع:**
  - هل خدمة الواجهة تعمل؟
  - هل خدمة الخادم تعمل؟
- **إجراء فوري:**
  - إعادة تشغيل الواجهة والخادم.
  - التحقق من المنافذ (`3001` للواجهة و`5000` للخادم).

### 2) فشل إنشاء الطلب

- **التحقق السريع:**
  - مراجعة سجل الأخطاء في الخادم.
  - التأكد من صحة البيانات المرسلة من صفحة الدفع.
- **إجراء فوري:**
  - تنفيذ طلب تجريبي ببيانات صحيحة.
  - التحقق من ظهور الطلب في لوحة الإدارة (`/admin/orders`).

### 3) أحداث التحليلات لا تصل

- **التحقق السريع:**
  - فحص وجود `googleAnalyticsId` و`tagManagerId` و`metaPixelId`.
  - التأكد من ظهور `window.dataLayer` في المتصفح.
- **إجراء فوري:**
  - إعادة حفظ إعدادات التتبع من لوحة الإدارة.
  - متابعة الأحداث عبر GA4 DebugView وTag Assistant وMeta Pixel Helper.

### 4) بطء واضح في الصفحات

- **التحقق السريع:**
  - فحص صور المنتجات الكبيرة أو غير المحسّنة.
  - مراجعة أخطاء المتصفح الحرجة (Console Errors).
- **إجراء فوري:**
  - اختبار الصفحة في وضع التخفي بدون إضافات.
  - تأجيل أي إطلاق ترويجي كبير لحين استقرار الأداء.

### 5) خطوات ما بعد الاحتواء

- توثيق السبب الجذري (Root Cause).
- تسجيل الإجراء الوقائي لمنع التكرار.
- تحديث قائمة التحقق الأسبوعية/الشهرية إذا لزم.
